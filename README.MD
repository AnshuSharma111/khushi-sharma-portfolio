# Portfolio Site - Khushi Sharma

This project is the portfolio site for my sister, Khushi Sharma, a law student.
It's built using React primariy with no backend as the site is static. Follows a academic / professional tone.
---
# Setup
This is a simple React App. Follow the given steps:

1. Clone the repository

> git clone https://github.com/AnshuSharma111/khushi-sharma-portfolio.git

2. Install the dependencies

> npm install

3. Run the application

> npm start
